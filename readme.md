# ShopWithGolang

A mini E-commerce platform built for learning purposes using Golang.

## Features
- Product Management
- User Authentication
- Shopping Cart
- Order Processing

## Tech Stack
- **Language:** Go (Golang)
- **Database:** PostgreSQL / MySQL (Optional)
- **Framework:** Gin / Fiber (Optional)

## Getting Started

### Prerequisites
- Go 1.20+ installed

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/shopwithgolang.git
   ```
2. Install dependencies:
   ```bash
   go mod tidy
   ```
3. Run the application:
   ```bash
   go run main.go
   ```
